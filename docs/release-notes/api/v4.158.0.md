### Changed

- **IP Addresses List** ([GET /networking/ips](/docs/api/networking/ip-addresses-list/))

  - In addition to `region`, responses for this command can now also be filtered by the following parameters:
    * `address`
    * `prefix`
    * `rdns`
    * `type`
